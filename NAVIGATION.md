# 网站导航说明

## 页面结构

本网站包含两个主要页面，提供不同语言和品牌的电视展示：

### 🌍 主页面 - 全球品牌展示
- **文件**: `index.html`
- **语言**: 英文
- **内容**: 全球主流电视品牌（包含中国品牌）
- **特色**: 国际化界面，专业英文描述

### 🇨🇳 中文页面 - 中国品牌展示
- **文件**: `chinese-brands.html`
- **语言**: 中文
- **内容**: 专注于中国电视品牌
- **特色**: 中文界面，本地化体验

## 导航方式

### 1. 顶部导航栏
两个页面都包含顶部导航栏，可以轻松切换：
- **Global Brands** - 跳转到全球品牌页面
- **Chinese Brands** - 跳转到中国品牌页面

### 2. 直接访问
- 双击 `index.html` - 打开全球品牌主页面
- 双击 `chinese-brands.html` - 打开中国品牌页面

## 功能对比

| 功能 | 全球品牌页面 | 中国品牌页面 |
|------|-------------|-------------|
| 品牌数量 | 10个主要品牌 | 12个中国品牌 |
| 界面语言 | 英文 | 中文 |
| 搜索功能 | 多维度搜索 | 智能搜索 |
| 筛选功能 | 区域+分类筛选 | 分类筛选 |
| 统计信息 | 品牌/分类/国家 | 品牌/分类 |

## 使用建议

### 选择全球品牌页面，如果你：
- 需要了解国际电视品牌
- 希望获得英文界面体验
- 需要按地区筛选品牌
- 想要更全面的品牌覆盖

### 选择中国品牌页面，如果你：
- 主要关注中国电视品牌
- 偏好中文界面
- 需要中文客服联系方式
- 想要更详细的中国品牌信息
